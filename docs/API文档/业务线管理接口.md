# 业务线管理接口

## 创建业务线

创建一个新的业务线。

### 请求

```
POST /api/v1/system/business
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 请求体

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 业务线名称 |
| code | string | 是 | 业务线编码（唯一） |
| icon | string | 否 | 业务线图标 |
| description | string | 否 | 业务线描述 |

### 响应

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "name": "业务线A",
    "code": "business_a",
    "icon": null,
    "description": null,
    "status": 1
  }
}
```

## 获取业务线列表

获取业务线列表（分页）。

### 请求

```
GET /api/v1/system/business?page=1&pageSize=10&status=1&keyword=
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 查询参数

| 参数 | 类型 | 说明 |
|------|------|------|
| page | number | 页码（默认 1） |
| pageSize | number | 每页数量（默认 10） |
| status | number | 状态：1-启用，0-禁用 |
| keyword | string | 搜索关键词（名称或编码） |

### 响应

```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": 1,
        "name": "业务线A",
        "code": "business_a",
        "icon": null,
        "description": null,
        "status": 1,
        "sort": 0,
        "createdAt": "2026-01-01T00:00:00.000Z"
      }
    ],
    "total": 1,
    "page": 1,
    "pageSize": 10
  }
}
```

### 请求头


### 路径参数

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 业务线ID |

### 响应

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "name": "业务线A",
    "code": "business_a",
    "icon": null,
    "description": null,
    "status": 1,
    "sort": 0,
    "createdAt": "2026-01-01T00:00:00.000Z",
    "updatedAt": "2026-01-01T00:00:00.000Z"
  }
}
```

## 获取业务下的应用列表

获取业务线下的所有应用模块。

### 请求

```
GET /api/v1/system/business/:id/apps
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 路径参数

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 业务线ID |

### 响应

```json
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "name": "Web前端",
      "code": "web_frontend",
      "type": 1,
      "description": "PC端管理系统",
      "logo": null,
      "domain": null
    }
  ]
}
```

## 应用类型说明

| 类型 | 说明 |
|------|------|
| 1 | PC端 |
| 2 | Mobile端 |
| 3 | 小程序 |

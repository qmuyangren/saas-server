# 应用模块管理接口

## 创建应用模块

为业务线创建一个新的应用模块。

### 请求

```
POST /api/v1/system/app
```

### 请求头

```
Authorization: Bearer <access_token>
Permission: app:create
```

### 请求体

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| businessId | number | 是 | 业务线ID |
| name | string | 是 | 应用名称 |
| code | string | 是 | 应用编码（唯一） |
| type | number | 是 | 应用类型（1-PC端，2-Mobile端，3-小程序） |
| description | string | 否 | 应用描述 |
| logo | string | 否 | 应用图标 |
| domain | string | 否 | 应用域名 |

### 响应

```json
{
  "code": 200,
  "data": {
    "success": true,
    "appId": 1
  }
}
```

## 获取应用列表

获取应用模块列表（分页）。

### 请求

```
GET /api/v1/system/app?page=1&pageSize=10&status=1&keyword=
```

### 请求头

```
Authorization: Bearer <access_token>
Permission: app:read
```

### 查询参数

| 参数 | 类型 | 说明 |
|------|------|------|
| page | number | 页码（默认 1） |
| pageSize | number | 每页数量（默认 10） |
| status | number | 状态：1-启用，0-禁用 |
| keyword | string | 搜索关键词（名称或编码） |
| businessId | number | 业务线ID（可选） |

### 响应

```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": 1,
        "name": "Web前端",
        "code": "web_frontend",
        "type": 1,
        "description": "PC端管理系统",
        "logo": null,
        "domain": null,
        "businessId": 1,
        "status": 1,
        "sort": 0,
        "createdAt": "2026-01-01T00:00:00.000Z"
      }
    ],
    "total": 1,
    "page": 1,
    "pageSize": 10
  }
}
```

## 更新应用

更新应用信息。

### 请求

```
PUT /api/v1/system/app/:id
```

### 请求头

```
Authorization: Bearer <access_token>
Permission: app:update
```

### 路径参数

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 应用ID |

### 请求体

| 参数 | 类型 | 说明 |
|------|------|------|
| name | string | 应用名称 |
| type | number | 应用类型 |
| description | string | 应用描述 |
| logo | string | 应用图标 |
| domain | string | 应用域名 |

### 响应

```json
{
  "code": 200,
  "message": "success"
}
```

### 请求头

```
Authorization: Bearer <access_token>
Permission: app:delete
```

### 路径参数

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 应用ID |

### 响应

```json
{
  "code": 200,
  "message": "success"
}
```

## 应用类型说明

| 类型 | 说明 |
|------|------|
| 1 | PC端应用 |
| 2 | Mobile端应用 |
| 3 | 小程序应用 |

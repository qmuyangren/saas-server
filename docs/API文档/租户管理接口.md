# 租户管理接口

## 创建租户

创建一个新的租户。

### 请求

```
POST /api/v1/system/tenant
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 请求体

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 租户名称 |
| code | string | 是 | 租户编码（唯一） |
| logo | string | 否 | 租户 logo |
| contactName | string | 否 | 联系人姓名 |
| contactPhone | string | 否 | 联系电话 |
| contactEmail | string | 否 | 联系邮箱 |
| domain | string | 否 | 租户专属域名 |
| maxUsers | number | 否 | 最大用户数（默认 100） |
| expireAt | string | 否 | 过期时间（ISO 8601） |
| remark | string | 否 | 备注 |

### 响应

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "name": "测试租户",
    "code": "test_tenant",
    "logo": null,
    "status": 1
  }
}
```

## 获取租户列表

获取租户列表（分页）。

### 请求

```
GET /api/v1/system/tenant?page=1&pageSize=10&status=1&keyword=
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 查询参数

| 参数 | 类型 | 说明 |
|------|------|------|
| page | number | 页码（默认 1） |
| pageSize | number | 每页数量（默认 10） |
| status | number | 状态：1-启用，0-禁用 |
| keyword | string | 搜索关键词（名称或编码） |

### 响应

```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": 1,
        "name": "默认租户",
        "code": "default",
        "logo": null,
        "contactName": null,
        "contactPhone": null,
        "contactEmail": null,
        "domain": null,
        "maxUsers": 100,
        "usedUsers": 0,
        "status": 1,
        "expireAt": null,
        "remark": null,
        "createdAt": "2026-01-01T00:00:00.000Z"
      }
    ],
    "total": 1,
    "page": 1,
    "pageSize": 10
  }
}
```

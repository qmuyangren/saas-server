# 认证相关接口

## 获取 RSA 公钥

获取 RSA 公钥用于前端密码加密传输。

### 请求

```
GET /api/v1/system/rsa/public-key
```

### 响应

```json
{
  "code": 200,
  "data": {
    "publicKey": "-----BEGIN PUBLIC KEY-----\nMIIB...END PUBLIC KEY-----\n"
  }
}
```

## 账密登录

使用账号密码登录系统，支持 RSA 加密密码。

### 请求

```
POST /api/v1/system/login/account
```

### 请求体

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| account | string | 是 | 用户名或邮箱 |
| password | string | 是 | 密码（支持 RSA 加密） |
| clientId | string | 是 | 客户端ID |
| deviceId | string | 否 | 设备ID |
| deviceName | string | 否 | 设备名称 |
| captchaId | string | 否 | 验证码ID |
| captchaValue | string | 否 | 验证码值 |
| tenantId | string | 否 | 租户ID（单租户可选） |

### 响应

```json
{
  "code": 200,
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIs...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
      "id": "1",
      "username": "admin",
      "nickname": "管理员",
      "avatar": ""
    },
    "tenants": [
      {
        "id": 1,
        "name": "默认租户",
        "code": "default",
        "logo": null,
        "isDefault": 1
      }
    ],
    "needSelectTenant": false,
    "defaultTenant": {
      "id": 1,
      "name": "默认租户",
      "code": "default",
      "logo": null,
      "isDefault": 1
    }
  }
}
```

## 刷新 Token

使用 Refresh Token 获取新的 Access Token。

### 请求

```
POST /api/v1/system/token/refresh
```

### 请求头

```
Authorization: Bearer <refresh_token>
```

### 请求体

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| clientId | string | 是 | 客户端ID |
| deviceId | string | 否 | 设备ID |

### 响应

```json
{
  "code": 200,
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIs...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIs..."
  }
}
```

## 退出登录

注销当前 Token（加入黑名单）。

### 请求

```
POST /api/v1/system/logout
```

### 请求头

```
Authorization: Bearer <access_token>
```

### 响应

```json
{
  "code": 200,
  "message": "success"
}
```

## 获取系统配置

获取当前客户端的系统配置（如登录方式、验证码设置等）。

### 请求

```
GET /api/v1/system/config?clientId=admin_web
```

### 响应

```json
{
  "code": 200,
  "data": {
    "name": "后台管理前端",
    "logo": null,
    "themeColor": "#0052d9",
    "enableLoginMethods": ["account", "wechat"],
    "enableCaptcha": true,
    "captchaType": "slider"
  }
}
```

## 错误码

| 错误码 | 说明 |
|--------|------|
| 400 | 客户端错误（参数错误、用户不存在、密码错误等） |
| 401 | 未授权（Token 无效或已过期） |
| 403 | 禁止访问（无权限） |
| 500 | 服务器内部错误 |

## JWT Token 说明

### Access Token

- 有效期：2 小时
- 用途：访问需要认证的接口
- 存放在：请求头 `Authorization: Bearer <token>`

### Refresh Token

- 有效期：7 天
- 用途：获取新的 Access Token
- 存放在：本地存储，定期刷新

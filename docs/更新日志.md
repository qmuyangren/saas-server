### 🔧 技术细节

#### RSA 加密流程
```
1. 前端加载页面时调用 /system/rsa/public-key 获取公钥
2. 用户输入密码后，使用公钥加密（雪花加密）
3. 后端接收到加密密码，使用私钥解密
4. 解密后的密码使用 bcrypt 进行哈希校验
```

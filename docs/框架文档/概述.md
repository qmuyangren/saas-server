# 框架概述

## 项目介绍

企业级管理系统后端服务，基于 NestJS 框架构建，采用多租户 SaaS 架构设计。

## 核心特性

- **多租户支持** - 用户可属于多个租户，租户间数据隔离
- **多业务线** - 租户可订阅多个业务，业务间配置独立
- **应用模块化** - 每个业务下可包含多个应用模块（PC端、移动端、小程序）
- **RBAC权限** - 基于角色的权限控制，支持模块权限和数据权限
- **统一认证** - JWT Token 认证，支持 RSA 密钥对加密
- **滑块验证码** - 防止暴力破解，提升安全性

## 数据库模型

主要模型：
- **BaseUser** - 用户表
- **Tenant** - 租户表
- **Business** - 业务线表
- **App** - 应用模块表
- **BaseRole** - 角色表
- **BasePermission** - 权限表

完整模型请查看 [prisma/schema.prisma](../prisma/schema.prisma)

## 二、源代码目录结构标准

### 2.1 目录组织原则

```
src/
├── main.ts                          # 应用入口（必须）
├── app.module.ts                    # 根模块（必须）
├── config/                          # 配置目录（必须）
│   ├── configuration.ts            # 配置加载主文件
│   ├── validation.schema.ts        # 环境变量验证
│   └── environments/               # 环境特定配置
│       ├── development.config.ts
│       ├── production.config.ts
│       └── test.config.ts
│
├── modules/                         # 业务模块目录（必须）
│   └── <module_name>/              # 按业务领域划分
│       ├── <module>.module.ts      # 模块定义
│       ├── controllers/            # 控制器层（必须）
│       ├── services/               # 服务层（必须）
│       ├── repositories/           # 仓储层（必须）
│       └── dto/                    # 数据传输对象（必须）
│           ├── requests/           # 请求 DTO
│           └── responses/          # 响应 DTO
│
├── common/                          # 通用组件目录（必须）
│   ├── decorators/                 # 自定义装饰器
│   ├── filters/                    # 异常过滤器
│   ├── interceptors/               # 拦截器
│   ├── guards/                     # 守卫
│   ├── middleware/                 # 中间件
│   ├── pipes/                      # 管道
│   ├── interfaces/                 # 通用接口
│   ├── constants/                  # 常量定义
│   ├── exceptions/                 # 自定义异常
│   └── utils/                      # 工具函数
│
├── infrastructure/                  # 基础设施目录（必须）
│   ├── database/                   # 数据库基础设施
│   ├── cache/                      # 缓存基础设施
│   ├── queue/                      # 消息队列
│   ├── http/                       # HTTP 客户端
│   ├── storage/                    # 文件存储
│   └── lock/                       # 分布式锁
│
└── types/                           # 类型定义目录（必须）
    ├── global.d.ts                 # 全局类型扩展
    ├── express.d.ts                # Express 类型扩展
    └── prisma.d.ts                 # Prisma 类型扩展
```

### 2.2 目录职责说明

| 目录 | 职责 | 被依赖方向 | 是否可空 |
|------|------|------------|----------|
| `modules/` | 业务模块，按领域划分 | 可依赖 common、infrastructure、config | ❌ 否 |
| `common/` | 通用组件，跨模块共享 | 不依赖 modules、infrastructure | ❌ 否 |
| `infrastructure/` | 基础设施，封装外部系统 | 不依赖 modules 和 common | ❌ 否 |
| `config/` | 应用配置管理 | 被所有模块依赖 | ❌ 否 |
| `types/` | 全局类型定义 | 被所有模块依赖 | ❌ 否 |
| `prisma/` | 数据模型和迁移 | 被 infrastructure/database 依赖 | ❌ 否 |
| `test/` | 测试代码 | 依赖 src | ✅ 是 |
| `scripts/` | 运维脚本 | 独立运行 | ✅ 是 |

## 三、模块开发标准

### 3.1 模块最小结构

每个业务模块必须包含以下文件：

```bash
modules/<module>/
├── <module>.module.ts          # 模块定义（必须）
├── controllers/                # 控制器目录（必须）
│   └── <module>.controller.ts
├── services/                   # 服务层目录（必须）
│   └── <module>.service.ts
├── repositories/               # 仓储层目录（必须）
│   └── <module>.repository.ts
└── dto/                        # DTO 目录（必须）
    ├── requests/               # 请求 DTO（必须）
    └── responses/              # 响应 DTO（必须）
```

### 3.2 模块依赖原则

```
✅ 允许的依赖方向：
modules/  →  common/  →  infrastructure/
          ↓
        config/
          ↓
        types/

❌ 禁止的依赖方向：
modules/  →  infrastructure/
modules/  →  common/  (仅当通用时)
infrastructure/  →  modules/
```

### 3.3 模块示例

```typescript
// modules/user/user.module.ts
import { Module } from '@nestjs/common';
import { PrismaModule } from '@/infrastructure/database/prisma.module';
import { UserController } from './controllers/user.controller';
import { UserService } from './services/user.service';
import { UserRepository } from './repositories/user.repository';

@Module({
  imports: [PrismaModule],
  controllers: [UserController],
  providers: [UserService, UserRepository],
  exports: [UserService, UserRepository], // 如需被其他模块使用
})
export class UserModule {}
```

## 四、数据库模型标准

### 4.1 Prisma Schema 规范

#### 基础模型定义

```prisma
/// 模型说明（必填）
/// 描述：模型的具体作用和业务含义
/// 作者：所有人
/// 创建时间：YYYY-MM-DD

model ModelName {
  id          BigInt      @id @default(autoincrement()) @map("id")
  uuid        String      @unique @map("uuid") @db.VarChar(32)
  
  // 业务字段...
  
  status      Int         @default(1) @map("status")        // 状态：0-禁用 1-启用
  version     Int         @default(1) @map("version")        // 乐观锁版本
  
  // 审计字段（必须）
  createdBy   BigInt?     @map("created_by")
  createdAt   DateTime    @default(now()) @map("created_at")
  updatedBy   BigInt?     @map("updated_by")
  updatedAt   DateTime    @updatedAt @map("updated_at")
  isDeleted   Int         @default(0) @map("is_deleted")     // 软删除标记
  deletedBy   BigInt?     @map("deleted_by")
  deletedAt   DateTime?   @map("deleted_at")
  
  // 关系引用
  relationField RelatedModel? @relation("RelationName", fields: [fid], references: [id])
  
  @@index([fieldName])
  @@map("table_name")
}
```

#### 字段命名规范

| 字段类型 | 命名规范 | 示例 | 类型 |
|----------|----------|------|------|
| 主键 | `id` | `id` | `BigInt` |
| 唯一标识 | `uuid` | `uuid` | `String(32)` |
| 状态 | `status` | `status` | `Int` (0-禁用, 1-启用) |
| 创建时间 | `createdAt` | `created_at` | `DateTime` |
| 更新时间 | `updatedAt` | `updated_at` | `DateTime` |
| 创建人 | `createdBy` | `created_by` | `BigInt?` |
| 更新人 | `updatedBy` | `updated_by` | `BigInt?` |
| 软删除 | `isDeleted` | `is_deleted` | `Int` (0-否, 1-是) |
| 删除人 | `deletedBy` | `deleted_by` | `BigInt?` |
| 删除时间 | `deletedAt` | `deleted_at` | `DateTime?` |
| 版本 | `version` | `version` | `Int` |

#### 索引规范

```prisma

// 唯一索引

// 普通索引
@@index([fieldName])

// 复合索引
@@index([field1, field2])

// 外键索引（通过 @relation 自动创建）
```

## 五、API 设计规范

### 5.1 RESTful API 规范

#### 路由设计

| 操作 | HTTP Method | 路径 | 说明 |
|------|-------------|------|------|
| 查询列表 | GET | `/api/v1/<resource>` | 支持分页、筛选、排序 |
| 查询详情 | GET | `/api/v1/<resource>/{id}` |  |
| 创建资源 | POST | `/api/v1/<resource>` |  |
| 更新资源 | PUT | `/api/v1/<resource>/{id}` | 全量更新 |
| 部分更新 | PATCH | `/api/v1/<resource>/{id}` |  |
| 删除资源 | DELETE | `/api/v1/<resource>/{id}` | 软删除 |
| 批量操作 | POST | `/api/v1/<resource>/batch` |  |

#### Controller 示例

```typescript
// controllers/user.controller.ts
import { Controller, Get, Post, Put, Delete, Param, Body, Query, UseGuards } from '@nestjs/common';
import { UserService } from '../services/user.service';
import { AuthGuard } from '@/common/guards/auth.guard';
import { CurrentUser } from '@/common/decorators/current-user.decorator';
import { CreateUserDto } from '../dto/requests/create-user.dto';
import { UpdateUserDto } from '../dto/requests/update-user.dto';

@Controller('api/v1/users')
@UseGuards(AuthGuard)
export class UserController {
  constructor(private readonly userService: UserService) {}

  // GET /api/v1/users?keyword=xxx&page=1&limit=20
  @Get()
  async findAll(@Query() query: any, @CurrentUser() user: any) {
    return this.userService.findAll(query);
  }

  // GET /api/v1/users/{id}
  @Get(':id')
  async findOne(@Param('id') id: number) {
    return this.userService.findById(id);
  }

  // POST /api/v1/users
  @Post()
  async create(@Body() dto: CreateUserDto) {
    return this.userService.create(dto);
  }

  // PUT /api/v1/users/{id}
  @Put(':id')
  async update(@Param('id') id: number, @Body() dto: UpdateUserDto) {
    return this.userService.update(id, dto);
  }

  // DELETE /api/v1/users/{id}
  @Delete(':id')
  async remove(@Param('id') id: number) {
    return this.userService.remove(id);
  }
}
```

## 七、代码规范

### 7.1 Service 层规范

```typescript
// services/user.service.ts
import { Injectable, BusinessException } from '@/common';
import { UserRepository } from '../repositories/user.repository';
import { CreateUserDto } from '../dto/requests/create-user.dto';

@Injectable()
export class UserService {
  constructor(private readonly userRepository: UserRepository) {}

  /**
   * 根据 ID 查询用户
   * @param id 用户 ID
   */
  async findById(id: number): Promise<any> {
    const user = await this.userRepository.findById(id);
    if (!user) {
      throw new BusinessException(2002, '用户不存在');
    }
    return user;
  }

  /**
   * 创建用户
   * @param dto 创建用户 DTO
   */
  async create(dto: CreateUserDto): Promise<any> {
    // 检查用户名是否已存在
    const exists = await this.userRepository.exists({
      where: { username: dto.username },
    });
    if (exists) {
      throw new BusinessException(2001, '用户名已存在');
    }

    // 创建用户
    return this.userRepository.create({
      ...dto,
      password: this.cryptoService.hash(dto.password),
    });
  }

  /**
   * 更新用户
   * @param id 用户 ID
   * @param dto 更新数据
   */
  async update(id: number, dto: any): Promise<any> {
    const user = await this.findById(id);
    return this.userRepository.update(id, dto);
  }

  /**
   * 删除用户（软删除）
   * @param id 用户 ID
   */
  async remove(id: number): Promise<void> {
    const user = await this.findById(id);
    await this.userRepository.softDelete(id);
  }
}
```

### 7.2 Repository 层规范

```typescript
// repositories/user.repository.ts
import { Injectable } from '@nestjs/common';
import { BaseRepository } from '@/infrastructure/database/base.repository';
import { PrismaService } from '@/infrastructure/database/prisma.service';

@Injectable()
export class UserRepository extends BaseRepository {
  constructor(prisma: PrismaService) {
    super(prisma, 'baseUser');
  }

  /**
   * 根据邮箱查找用户
   */
  async findByEmail(email: string): Promise<any> {
    return this.prisma.baseUser.findFirst({
      where: { email },
    });
  }

  /**
   * 根据用户名查找用户
   */
  async findByUsername(username: string): Promise<any> {
    return this.prisma.baseUser.findFirst({
      where: { username },
    });
  }

  /**
   * 分页查询用户（含关联信息）
   */
  async findWithPagination(query: any): Promise<any> {
    const { keyword, status, page = 1, limit = 20 } = query;
    
    const where: any = {
      isDeleted: 0,
    };

    if (keyword) {
      where.OR = [
        { username: { contains: keyword } },
        { nickname: { contains: keyword } },
        { email: { contains: keyword } },
      ];
    }

    if (status !== undefined) {
      where.status = status;
    }

    return this.findAll({ where, page, limit });
  }
}
```

### 8.5 配置加载与使用

#### 配置服务接口

```typescript
// config/services/config.service.ts
import { Injectable, BusinessException } from '@/common';
import { InjectModel } from '@nestjs/prisma';
import { PrismaService } from '@/infrastructure/database/prisma.service';
import { ConfigService as NestConfigService } from '@nestjs/config';

@Injectable()
export class ConfigService {
  constructor(
    private readonly prisma: PrismaService,
    private readonly nestConfig: NestConfigService,
  ) {}

  /**
   * 获取系统配置
   * @param key 配置键
   * @param defaultValue 默认值
   */
  async get<string>(key: string, defaultValue?: string): Promise<string | null> {
    const config = await this.prisma.cfgSystemConfig.findUnique({
      where: { configKey: key },
    });
    
    if (!config && defaultValue !== undefined) {
      return defaultValue;
    }
    
    return config?.configValue ?? null;
  }

  /**
   * 获取 JSON 类型配置
   * @param key 配置键
   * @param defaultValue 默认值
   */
  async getJson<T = any>(key: string, defaultValue?: T): Promise<T> {
    const value = await this.get(key);
    if (!value) {
      return defaultValue as T;
    }
    try {
      return JSON.parse(value);
    } catch {
      return defaultValue as T;
    }
  }

  /**
   * 获取第三方服务配置
   * @param provider 服务商代码
   * @param type 配置类型
   */
  async getThirdPartyConfig(
    provider: string,
    type: string,
  ): Promise<any> {
    const config = await this.prisma.cfgThirdParty.findFirst({
      where: { code: provider, type },
    });
    
    if (!config) {
      throw new BusinessException(2002, `服务商配置不存在: ${provider}`);
    }
    
    try {
      return JSON.parse(config.config);
    } catch {
      throw new BusinessException(5000, '服务商配置格式错误');
    }
  }
}
```

### 8.6 配置缓存策略

```typescript
// modules/config/services/config.cache.service.ts
import { Injectable } from '@nestjs/common';
import { RedisService } from '@/infrastructure/cache/redis.service';
import { ConfigService } from './config.service';

@Injectable()
export class ConfigCacheService {
  private readonly CACHE_PREFIX = 'config:';
  private readonly CACHE_TTL = 3600; // 1小时

  constructor(
    private readonly redis: RedisService,
    private readonly config: ConfigService,
  ) {}

  async getWithCache<string>(key: string): Promise<string | null> {
    const cacheKey = `${this.CACHE_PREFIX}${key}`;
    const cached = await this.redis.get<string>(cacheKey);
    
    if (cached) {
      return cached;
    }
    
    const value = await this.config.get<string>(key);
    if (value) {
      await this.redis.set(cacheKey, value, this.CACHE_TTL);
    }
    
    return value;
  }

  async getJsonWithCache<T = any>(key: string): Promise<T> {
    const value = await this.getWithCache(key);
    if (!value) {
      return null;
    }
    try {
      return JSON.parse(value);
    } catch {
      return null;
    }
  }

  async invalidateCache(key?: string): Promise<void> {
    if (key) {
      await this.redis.del(`${this.CACHE_PREFIX}${key}`);
    } else {
      // 清除所有配置缓存
      const keys = await this.redis.keys(`${this.CACHE_PREFIX}*`);
      if (keys.length > 0) {
        await this.redis.del(keys);
      }
    }
  }
}
```

```typescript
import { ConfigService } from '@nestjs/config';
import { Request } from 'express';

@Injectable()
export class IpWhitelistGuard implements CanActivate {
  constructor(private readonly config: ConfigService) {}

  canActivate(context: ExecutionContext): boolean {
    const request = context.switchToHttp().getRequest<Request>();
    const clientIp = this.getClientIp(request);
    const allowedIps = this.config.get<string[]>(
      'security.allowed_ips',
      [],
    );

    // 如果白名单为空，允许所有
    if (allowedIps.length === 0) {
      return true;
    }
```

### 13.6 通讯服务实现示例

```typescript
// modules/message/services/message.service.ts
import { Injectable, BusinessException } from '@/common';
import { Cron, CronExpression } from '@nestjs/schedule';
import { ConfigService } from '@nestjs/config';
import { PrismaService } from '@/infrastructure/database/prisma.service';
import { RedisService } from '@/infrastructure/cache/redis.service';

@Injectable()
export class MessageService {
  constructor(
    private readonly config: ConfigService,
    private readonly prisma: PrismaService,
    private readonly redis: RedisService,
  ) {}

  private readonly CACHE_PREFIX = 'message:ratelimit:';
  private readonly CACHE_TTL = 60; // 1分钟

  /**
   * 发送站内消息
   */
  async sendInAppMessage(userId: number, title: string, content: string): Promise<void> {
    await this.prisma.inAppMessage.create({
      data: { userId, title, content },
    });
  }

  /**
   * 批量发送站内消息
   */
  async sendBatchInAppMessage(userIds: number[], title: string, content: string): Promise<void> {
    const config = this.config.getJson('message.in_app');
    if (userIds.length > config.batch_send_limit) {
      throw new BusinessException(1000, `批量发送人数超过限制: ${config.batch_send_limit}`);
    }

    await this.prisma.inAppMessage.createMany({
      data: userIds.map(userId => ({ userId, title, content })),
    });
  }

  /**
   * 发送短信
   */
  async sendSms(phone: string, templateCode: string, params: any): Promise<void> {
    const config = this.config.getJson('message.sms');
    if (!config.enabled) {
      return;
    }

    // 检查频率限制
    await this.checkRateLimit('sms', phone);

    // 调用短信服务商 API
    if (config.provider === 'aliyun') {
      await this.aliyunSms.send(phone, templateCode, params);
    } else if (config.provider === 'tencent') {
      await this.tencentSms.send(phone, templateCode, params);
    }
  }

  /**
   * 发送邮件
   */
  async sendEmail(email: string, templateId: string, params: any): Promise<void> {
    const config = this.config.getJson('message.email');
    if (!config.enabled) {
      return;
    }

    // 检查频率限制
    await this.checkRateLimit('email', email);

    // 获取模板内容
    const template = await this.getEmailTemplate(templateId);
    const subject = this.renderTemplate(template.subject, params);
    const body = this.renderTemplate(template.content, params);

    // 发送邮件
    await this.mailer.sendMail({
      to: email,
      subject,
      html: body,
    });
  }

  /**
   * 发送微信模板消息
   */
  async sendWechatTemplate(openid: string, templateId: string, params: any): Promise<void> {
    const config = this.config.getJson('message.wechat');
    if (!config.enabled) {
      return;
    }

    // 调用微信 API
    await this.wechatMessage.sendTemplate(openid, templateId, params);
  }

  /**
   * 消息入队（异步发送）
   */
  async queueMessage(userId: number, type: string, title: string, content: string): Promise<void> {
    await this.queue.add('message-send', {
      userId,
      type,
      title,
      content,
    }, { attempts: 3, backoff: 5000 });
  }

  /**
   * 消息发送处理器（定时任务）
   */
  @Cron(CronExpression.EVERY_MINUTE)
  async processQueue() {
    const message = await this.queue.getNext('message-send');
    if (message) {
      try {
        await this.sendMessage(message.data);
      } catch (error) {
        this.logger.error('消息发送失败', error);
      }
    }
  }

  /**
   * 频率限制检查
   */
  private async checkRateLimit(channel: string, target: string): Promise<void> {
    const limitConfig = this.config.getJson(`message.${channel}.rate_limit`);
    const key = `${this.CACHE_PREFIX}${channel}:${target}`;
    const current = await this.redis.incr(key);
    if (current === 1) {
      await this.redis.expire(key, this.CACHE_TTL);
    }
    if (current > limitConfig.per_minute) {
      throw new BusinessException(1008, '发送频率过高');
    }
  }
}
```

### 14.2 权限校验守卫

#### 权限校验守卫接口
```typescript
// common/guards/permission.guard.ts
import {
  Injectable,
  CanActivate,
  ExecutionContext,
  ForbiddenException,
} from '@nestjs/common';
import { Reflector } from '@nestjs/core';
import { PERMISSION_KEY } from '@/common/decorators/permission.decorator';

@Injectable()
export class PermissionGuard implements CanActivate {
  constructor(private readonly reflector: Reflector) {}

  canActivate(context: ExecutionContext): boolean {
    const requiredPermissions = this.reflector.getAllAndMerge<string[]>(
      PERMISSION_KEY,
      [context.getHandler(), context.getClass()],
    );

    if (!requiredPermissions || requiredPermissions.length === 0) {
      return true; // 没有权限要求，放行
    }

    const request = context.switchToHttp().getRequest();
    const user = request.user;

    // 检查用户是否拥有必需的权限
    const hasPermission = this.checkPermissions(user, requiredPermissions);
    if (!hasPermission) {
      throw new ForbiddenException({
        code: 1002,
        message: '权限不足',
      });
    }

    return true;
  }

  private checkPermissions(user: any, requiredPermissions: string[]): boolean {
    if (!user || !user.permissions) {
      return false;
    }

    // 检查用户权限列表是否包含必需权限
    return requiredPermissions.every(permission => 
      user.permissions.includes(permission)
    );
  }
}
```

```typescript
// common/guards/data-permission.guard.ts
import {
  Injectable,
  CanActivate,
  ExecutionContext,
  ForbiddenException,
} from '@nestjs/common';
import { Reflector } from '@nestjs/core';
import { DATA_PERMISSION_KEY } from '@/common/decorators/data-permission.decorator';

export enum DataPermissionType {
  ALL = 'all',        // 全部数据
  THIS = 'this',      // 仅本人数据
  DEPARTMENT = 'department',  // 本部门数据
  UNDER = 'under',    // 下属部门数据
  CUSTOM = 'custom',  // 自定义数据
}

@Injectable()
export class DataPermissionGuard implements CanActivate {
  constructor(private readonly reflector: Reflector) {}

  canActivate(context: ExecutionContext): boolean {
    const permissionType = this.reflector.getAllAndMerge<DataPermissionType>(
      DATA_PERMISSION_KEY,
      [context.getHandler(), context.getClass()],
    );

    if (!permissionType) {
      return true; // 没有数据权限要求
    }

    const request = context.switchToHttp().getRequest();
    const user = request.user;

    // 根据权限类型添加数据过滤条件
    request.dataPermission = this.buildDataPermissionFilter(user, permissionType);

    return true;
  }

  private buildDataPermissionFilter(user: any, type: DataPermissionType): any {
    switch (type) {
      case DataPermissionType.THIS:
        return { userId: user.id };
      
      case DataPermissionType.DEPARTMENT:
        return { departmentId: user.departmentId };
      
      case DataPermissionType.UNDER:
        return { departmentIdIn: user.subDepartmentIds };
      
      case DataPermissionType.CUSTOM:
        return { ...user.customPermissionFilter };
      
      case DataPermissionType.ALL:
      default:
        return {}; // 无过滤条件
    }
  }
}
```

```typescript
import {

} from '@nestjs/common';

import {map} from 'rxjs/operators';

@Injectable()
export class OperateLogInterceptor implements NestInterceptor {
  intercept(context: ExecutionContext, next: CallHandler): Observable<any> {
    const request = context.switchToHttp().getRequest();
    const user = request.user;

    const startTime = Date.now();

    return next.handle().pipe(
      map(data => {
        const endTime = Date.now();
        
        // 记录操作日志
        this.logOperation(user, request, data, endTime - startTime);
        
        return data;
      }),
    );
  }

    const logData = {
      userId: user?.id,
      username: user?.username,
      module: this.getModule(request),
      operation: this.getOperation(request),
      method: request.method,
      url: request.url,
      params: this.maskParams(request.body),
      result: this.formatResult(result),
      status: result?.code === 200 ? 1 : 0,
      errorMsg: result?.code !== 200 ? result?.message : null,
      ip: request.ip,
      duration,
      userAgent: request.headers['user-agent'],
    };

    // 异步记录日志
    this.operateLogService.create(logData).catch(() => {});
  }

  private getModule(request: any): string {
    // 从路由或装饰器获取模块名
    return request.route?.config?.module || '未知模块';
  }

  private getOperation(request: any): string {
    // 根据HTTP方法和路由确定操作
    const methodMap: Record<string, string> = {
      POST: '创建',
      GET: '查询',
      PUT: '更新',
      DELETE: '删除',
      PATCH: '部分更新',
    };
    return methodMap[request.method] || '其他操作';
  }

  private maskParams(params: any): string {
    // 脱敏敏感字段
    if (!params) return '';
    const sensitiveKeys = ['password', 'secret', 'key', 'token'];
    return JSON.stringify(params, (key, value) => 
      sensitiveKeys.some(k => key.toLowerCase().includes(k)) ? '***' : value
    );
  }

  private formatResult(result: any): string {
    // 格式化返回结果
    return result?.code === 200 ? '成功' : `失败: ${result?.message}`;
  }
```

### 14.6 权限服务实现

```typescript
// modules/permission/services/permission.service.ts
import { Injectable, BusinessException } from '@/common';
import { PrismaService } from '@/infrastructure/database/prisma.service';

@Injectable()
export class PermissionService {
  constructor(private readonly prisma: PrismaService) {}

  /**
   * 获取用户权限列表（从权限组）
   */
  async getUserPermissions(userId: number): Promise<string[]> {
    const user = await this.prisma.baseUser.findUnique({
      where: { id: userId },
      include: {
        roles: true,
        department: true,
        position: true,
        groups: true,
      },
    });

    if (!user) {
      throw new BusinessException(2002, '用户不存在');
    }

    // 收集用户所属的所有角色、部门、岗位、分组的权限组ID
    const permissionGroupIds = new Set<number>();

    // 角色关联的权限组
    for (const userRole of user.roles) {
      const groups = await this.prisma.basePermissionGroup.findMany({
        where: {
          targets: {
            some: {
              targetType: 'role',
              targetId: userRole.roleId,
            },
          },
        },
      });
      groups.forEach(g => permissionGroupIds.add(g.id));
    }

    // 部门关联的权限组
    if (user.departmentId) {
      const deptGroups = await this.prisma.basePermissionGroup.findMany({
        where: {
          targets: {
            some: {
              targetType: 'department',
              targetId: user.departmentId,
            },
          },
        },
      });
      deptGroups.forEach(g => permissionGroupIds.add(g.id));
    }

    // 岗位关联的权限组
    if (user.positionId) {
      const posGroups = await this.prisma.basePermissionGroup.findMany({
        where: {
          targets: {
            some: {
              targetType: 'position',
              targetId: user.positionId,
            },
          },
        },
      });
      posGroups.forEach(g => permissionGroupIds.add(g.id));
    }

    // 分组关联的权限组
    for (const userGroup of user.groups) {
      const group = await this.prisma.baseGroup.findUnique({
        where: { id: userGroup.groupId },
        include: { targets: true },
      });
      group?.targets.forEach(t => {
        if (t.targetType === 'group') {
          permissionGroupIds.add(t.permissionGroupId);
        }
      });
    }

    // 获取所有权限组下的权限code
    const permissions = await this.prisma.basePermission.findMany({
      where: {
        permissionGroups: {
          some: {
            permissionGroupId: { in: Array.from(permissionGroupIds) },
          },
        },
      },
      select: { code: true },
    });

    return permissions.map(p => p.code);
  }

  /**
   * 获取菜单权限（树形结构）
   */
  async getMenuPermissions(userId: number): Promise<any[]> {
    const permissions = await this.getUserPermissions(userId);

    // 获取需要显示的菜单
    const menus = await this.prisma.basePermission.findMany({
      where: {
        type: 1, // 菜单类型
        code: { in: permissions },
        status: 1,
        parentId: 0,
      },
      orderBy: { sort: 'asc' },
      include: {
        children: {
          where: { status: 1 },
          orderBy: { sort: 'asc' },
        },
      },
    });

    return this.buildMenuTree(menus, permissions);
  }

  /**
   * 构建菜单树
   */
  private buildMenuTree(menus: any[], permissions: string[]): any[] {
    return menus.map(menu => {
      const menuCode = menu.code;
      const hasPermission = permissions.includes(menuCode);

      return {
        id: menu.id,
        name: menu.name,
        code: menuCode,
        path: menu.path,
        component: menu.component,
        icon: menu.icon,
        sort: menu.sort,
        children: menu.children.map(child => ({
          id: child.id,
          name: child.name,
          code: child.code,
          path: child.path,
          component: child.component,
          icon: child.icon,
          sort: child.sort,
          hasPermission: permissions.includes(child.code),
        })).filter(child => hasPermission || child.hasPermission),
        hasPermission,
      };
    });
  }

  /**
   * 分配权限组给对象
   */
  async assignPermissionGroupToTarget(
    permissionGroupId: number,
    targetType: string,
    targetId: number,
  ): Promise<void> {
    // 检查权限组是否存在
    const group = await this.prisma.basePermissionGroup.findUnique({
      where: { id: permissionGroupId },
    });
    if (!group) {
      throw new BusinessException(2002, '权限组不存在');
    }

    // 检查 target 是否存在
    let targetExists = false;
    switch (targetType) {
      case 'user':
        targetExists = !!(await this.prisma.baseUser.findUnique({
          where: { id: targetId },
        }));
        break;
      case 'role':
        targetExists = !!(await this.prisma.baseRole.findUnique({
          where: { id: targetId },
        }));
        break;
      case 'department':
        targetExists = !!(await this.prisma.baseDepartment.findUnique({
          where: { id: targetId },
        }));
        break;
      case 'position':
        targetExists = !!(await this.prisma.basePosition.findUnique({
          where: { id: targetId },
        }));
        break;
      case 'group':
        targetExists = !!(await this.prisma.baseGroup.findUnique({
          where: { id: targetId },
        }));
        break;
    }

    if (!targetExists) {
      throw new BusinessException(2002, '授权对象不存在');
    }

    // 创建授权关系
    await this.prisma.mapPermissionGroupTarget.create({
      data: {
        permissionGroupId,
        targetType,
        targetId,
      },
    });
  }

  /**
   * 撤销权限组授权
   */
  async revokePermissionGroupFromTarget(
    permissionGroupId: number,
    targetType: string,
    targetId: number,
  ): Promise<void> {
    await this.prisma.mapPermissionGroupTarget.deleteMany({
      where: {
        permissionGroupId,
        targetType,
        targetId,
      },
    });
  }
}
```

```typescript
          params.where = params.where || {};
          params.where.tenantId = tenantId;
        }
        return originalFindMany.call(request.prisma[request.model], params);
      };
```

```typescript
   * 租户升级
   */
  async upgrade(tenantId: number, dto: TenantUpgradeDto): Promise<void> {
    // 获取当前租户
    const tenant = await this.prisma.tenant.findUnique({
      where: { id: tenantId, isDeleted: 0 },
    });

    if (!tenant) {
      throw new BusinessException(2002, '租户不存在');

    });

    // 检查是否是相同套餐
    if (tenant.planCode === dto.planCode) {
      throw new BusinessException(1000, '当前已是该套餐');
    }

    // 更新租户套餐
    await this.prisma.tenant.update({
      where: { id: tenantId },
      data: {
        planCode: dto.planCode,
        planExpiredAt: this.calculateExpiredAt(tenant.planExpiredAt),
      },
    });

    // 记录升级日志
    await this.prisma.tenantUpgradeLog.create({
      data: {
        tenantId,
        fromPlanCode: tenant.planCode,
        toPlanCode: dto.planCode,
        upgradeType: 1, // 1-主动升级
        status: 1,      // 1-成功
        remark: dto.remark,
      },
    });
  }

   */
  private calculateExpiredAt(currentExpiredAt: Date | null): Date {

    const months = 12; // 默认一年
    const expiredAt = currentExpiredAt && currentExpiredAt > now 
      ? currentExpiredAt 
      : now;
    
    expiredAt.setMonth(expiredAt.getMonth() + months);

  }
```

```typescript
// modules/workflow/services/workflow.service.ts
import { Injectable, BusinessException } from '@/common';

@Injectable()
export class WorkflowService {
  /**
   * 启动流程
   */
  async startWorkflow(
    definitionCode: string,
    businessId: string,
    data: any,
    appId: number,
  ): Promise<WorkflowInstance> {
    // 获取流程定义
    const definition = await this.getDefinition(definitionCode);
    if (!definition) {
      throw new BusinessException(2002, '流程定义不存在');
    }

    if (definition.status !== 'active') {
      throw new BusinessException(1000, '流程未启用');
    }

    // 创建流程实例
    const instance = await this.prisma.workflowInstance.create({
      data: {
        definitionCode,
        businessId,
        data: JSON.stringify(data),
        currentNodeId: definition.nodes.find(n => n.type === 'start').id,
        status: 'running',
        appId,
      },
    });

    // 自动执行开始节点后的任务
    await this.executeNextTasks(instance.id);

    return instance;
  }

  /**
   * 执行下一个任务
   */
  private async executeNextTasks(instanceId: number): Promise<void> {
    const instance = await this.getInstance(instanceId);
    const definition = await this.getDefinition(instance.definitionCode);

    // 获取当前节点
    const currentNode = definition.nodes.find(n => n.id === instance.currentNodeId);
    if (!currentNode) {
      return;
    }

    // 判断节点类型
    switch (currentNode.type) {
      case 'task':
        // 创建任务
        await this.createTask(instanceId, currentNode);
        break;
      case 'gateway':
        // 网关分支处理
        await this.handleGateway(instanceId, currentNode, definition);
        break;
      case 'end':
        // 结束流程
        await this.endWorkflow(instanceId);
        break;
    }
  }

  /**
   * 审批任务
   */
  async approveTask(
    taskId: number,
    userId: number,
    approve: boolean,
    remarks: string,
  ): Promise<void> {
    const task = await this.getTask(taskId);
    if (!task) {
      throw new BusinessException(2002, '任务不存在');
    }

    if (task.status !== 'pending') {
      throw new BusinessException(1000, '任务已处理');
    }

    // 记录审批意见
    await this.prisma.workflowTaskHistory.create({
      data: {
        taskId,
        userId,
        approve,
        remarks,
        operatedAt: new Date(),
      },
    });

    // 更新任务状态
    await this.prisma.workflowTask.update({
      where: { id: taskId },
      data: {
        status: approve ? 'approved' : 'rejected',
        operatorId: userId,
        operatedAt: new Date(),
      },
    });

    // 更新流程实例
    const instance = await this.getInstance(task.workflowInstanceId);
    const nextNode = this.calculateNextNode(instance, approve);

    await this.prisma.workflowInstance.update({
      where: { id: instance.id },
      data: {
        currentNodeId: nextNode?.id,
        status: nextNode ? 'running' : 'completed',
      },
    });

    // 执行下一个节点
    if (nextNode) {
      await this.executeNextTasks(instance.id);
    }
  }

  /**
   * 计算下一个节点
   */
  private calculateNextNode(instance: any, approve: boolean): WorkflowNode | null {
    const definition = JSON.parse(instance.definition);
    const currentNode = definition.nodes.find(n => n.id === instance.currentNodeId);
    
    // 获取转移条件
    const transitions = definition.transitions.filter(t => t.from === currentNode.id);
    const matched = transitions.find(t => {
      if (!t.condition) return true;
      return this.evaluateCondition(t.condition, instance.data);
    });

    return matched ? definition.nodes.find(n => n.id === matched.to) : null;
  }

  /**
   * 评估条件表达式
   */
  private evaluateCondition(condition: string, data: any): boolean {
    // 实现条件表达式评估逻辑
    // 支持类似 "status == 'approved' && amount > 1000" 的表达式

  }

  /**
   * 结束流程
   */
  private async endWorkflow(instanceId: number): Promise<void> {
    await this.prisma.workflowInstance.update({
      where: { id: instanceId },
      data: {
        status: 'completed',
        completedAt: new Date(),
      },
    });
  }
```

```typescript
// modules/approval/services/approval.service.ts
import { Injectable, BusinessException } from '@/common';

@Injectable()
export class ApprovalService {
  /**
   * 提交审批申请
   */
  async submitApproval(
    businessId: string,
    businessType: string,
    data: any,
    applicantId: number,
  ): Promise<ApprovalInstance> {
    // 获取审批定义
    const definition = await this.getApprovalDefinition(businessType);
    if (!definition) {
      throw new BusinessException(2002, '审批流程未配置');
    }

    // 创建审批实例
    const instance = await this.prisma.approvalInstance.create({
      data: {
        businessId,
        businessType,
        data: JSON.stringify(data),
        applicantId,
        currentStepIndex: 0,
        status: 'pending',
      },
    });

    // 创建第一步审批任务
    const firstStep = definition.steps[0];
    await this.createApprovalTasks(instance.id, firstStep);

    return instance;
  }

  /**
   * 审批处理
   */
  async processApproval(
    taskId: number,
    userId: number,
    approve: boolean,
    remarks: string,
  ): Promise<void> {
    const task = await this.prisma.approvalTask.findUnique({
      where: { id: taskId },
      include: { instance: true },
    });

    if (!task) {
      throw new BusinessException(2002, '审批任务不存在');
    }

    // 记录审批记录
    await this.prisma.approvalRecord.create({
      data: {
        taskId: task.id,
        userId,
        approve,
        remarks,
        operatedAt: new Date(),
      },
    });

    // 更新任务状态
    await this.prisma.approvalTask.update({
      where: { id: taskId },
      data: {
        status: approve ? 'approved' : 'rejected',
        operatorId: userId,
        operatedAt: new Date(),
      },
    });

    // 检查是否全部完成
    const allTasks = await this.prisma.approvalTask.findMany({
      where: { instanceId: task.instanceId },
    });

    const allApproved = allTasks.every(t => t.status === 'approved');
    const anyRejected = allTasks.some(t => t.status === 'rejected');

    if (anyRejected) {
      // 驳回
      await this.prisma.approvalInstance.update({
        where: { id: task.instanceId },
        data: { status: 'rejected' },
      });
    } else if (allApproved) {
      // 检查是否还有下一步
      const instance = await this.prisma.approvalInstance.findUnique({
        where: { id: task.instanceId },
      });
      const definition = await this.getApprovalDefinition(instance.businessType);

      if (instance.currentStepIndex + 1 < definition.steps.length) {
        // 下一步
        const nextStep = definition.steps[instance.currentStepIndex + 1];
        await this.prisma.approvalInstance.update({
          where: { id: instance.id },
          data: { currentStepIndex: instance.currentStepIndex + 1 },
        });
        await this.createApprovalTasks(instance.id, nextStep);
      } else {
        // 完成
        await this.prisma.approvalInstance.update({
          where: { id: instance.id },
          data: { status: 'completed' },
        });
      }
    }
  }

  /**
   * 创建审批任务
   */
  private async createApprovalTasks(
    instanceId: number,
    step: ApprovalStep,
  ): Promise<void> {
    let taskType: 'sequential' | 'parallel' | 'or' = 'sequential';
    if (step.config.multiLevel) {
      taskType = step.type;
    }

    if (taskType === 'parallel' || taskType === 'or') {
      // 并行或或审批，同时创建多个任务
      for (const assigneeId of step.assignees) {
        await this.createSingleTask(instanceId, assigneeId, step);
      }
    } else {
      // 顺序审批，只创建一个任务
      const assigneeId = step.assignees[0];
      await this.createSingleTask(instanceId, assigneeId, step);
    }
  }

  /**
   * 创建单个审批任务
   */
  private async createSingleTask(
    instanceId: number,
```

```typescript
// modules/rule-engine/services/rule-engine.service.ts
import { Injectable, BusinessException } from '@/common';

@Injectable()
export class RuleEngineService {
  /**
   * 执行规则
   */
  async executeRules(
    businessType: string,
    data: any,
  ): Promise<RuleExecutionResult> {
    const rules = await this.getEnabledRules(businessType);
    
    // 按优先级排序
    rules.sort((a, b) => b.priority - a.priority);

    const results: RuleExecutionResultItem[] = [];
    let matchedCount = 0;

    for (const rule of rules) {
      const matched = this.evaluateRule(rule, data);
      
      if (matched) {
        matchedCount++;
        const actionResults = await this.executeActions(rule.actions, data);
        
        results.push({
          ruleId: rule.id,
          ruleCode: rule.code,
          ruleName: rule.name,
          matched: true,
          actionResults,
        });
      }
    }

    return {
      businessType,
      dataId: data.id,
      matchedCount,
      results,
    };
  }

  /**
   * 评估单条规则
   */
  private evaluateRule(rule: RuleDefinition, data: any): boolean {
    if (!rule.conditions || rule.conditions.length === 0) {
      return true;
    }

    let result = rule.conditions[0].type === 'and';
    for (const condition of rule.conditions) {
      const conditionMet = this.evaluateCondition(condition, data);
      
      if (condition.type === 'and') {
        result = result && conditionMet;
      } else {
        result = result || conditionMet;
      }
    }

    return result;
  }

  /**
   * 评估条件
   */
  private evaluateCondition(condition: Condition, data: any): boolean {
    const fieldValue = this.getNestedValue(data, condition.field);
    const compareValue = condition.value;

    switch (condition.operator) {
      case 'eq':
        return fieldValue == compareValue;
      case 'ne':
        return fieldValue != compareValue;
      case 'gt':
        return Number(fieldValue) > Number(compareValue);
      case 'lt':
        return Number(fieldValue) < Number(compareValue);
      case 'in':
        return Array.isArray(fieldValue) 
          ? fieldValue.some(v => compareValue.includes(v))
          : compareValue.includes(fieldValue);
      case 'not_in':
        return Array.isArray(fieldValue)
          ? !fieldValue.some(v => compareValue.includes(v))
          : !compareValue.includes(fieldValue);
      case 'contains':
        return String(fieldValue).includes(String(compareValue));
      case 'not_contains':
        return !String(fieldValue).includes(String(compareValue));
      default:
        return false;
    }
  }

  /**
   * 获取嵌套字段值
   */
  private getNestedValue(obj: any, path: string): any {
    return path.split('.').reduce((current, key) => current?.[key], obj);
  }

  /**
   * 执行规则动作
   */
  private async executeActions(actions: Action[], data: any): Promise<any[]> {
    const results = [];
    for (const action of actions) {
      let result: any;
      switch (action.type) {
        case 'assign':
          result = await this.assignAction(action.config, data);
          break;
        case 'notify':
          result = await this.notifyAction(action.config, data);
          break;
        case 'update':
          result = await this.updateAction(action.config, data);
          break;
        // ... 其他动作类型
        default:
          result = { success: false, message: '未知动作类型' };
      }
      results.push(result);
    }
    return results;
  }

}
```

```typescript
// modules/app/services/app.service.ts
import { Injectable, BusinessException } from '@/common';

@Injectable()
export class AppService {
  constructor(
    private readonly config: ConfigService,
    private readonly prisma: PrismaService,
  ) {}

  /**
   * 应用注册
   */
  async register(dto: AppRegisterDto, tenantId: number): Promise<AppEntity> {
    // 检查应用名称是否已存在
    const existsByName = await this.prisma.app.findFirst({
      where: { name: dto.name, tenantId, isDeleted: 0 },
    });
    if (existsByName) {
      throw new BusinessException(2001, '应用名称已存在');
    }

    // 检查应用代码是否已存在
    const existsByCode = await this.prisma.app.findFirst({
      where: { code: dto.code, tenantId, isDeleted: 0 },
    });
    if (existsByCode) {
      throw new BusinessException(2001, '应用标识已存在');
    }

    // 生成应用密钥
    const appKey = this.generateAppKey();
    const appSecret = this.generateAppSecret();

    // 创建应用
    const app = await this.prisma.app.create({
      data: {
        name: dto.name,
        code: dto.code,
        description: dto.description,
        callbackUrl: dto.callbackUrl,
        logo: dto.logo,
        website: dto.website,
        appKey,
        appSecret: this.cryptoService.hash(appSecret), // 加密存储
        featureFlags: dto.featureFlags,
        allowedOrigins: dto.allowedOrigins,
        tenantId,
        status: 1,
      },
    });

    return {
      ...app,
      appSecret, // 只返回一次，不存储明文
    };
  }

  /**
   * 刷新应用密钥
   */
  async refreshSecret(appId: number, tenantId: number): Promise<{ appKey: string; appSecret: string }> {
    const app = await this.prisma.app.findUnique({
      where: { id: appId, tenantId, isDeleted: 0 },
    });

    if (!app) {
      throw new BusinessException(2002, '应用不存在');
    }

    const newAppKey = this.generateAppKey();
    const newAppSecret = this.generateAppSecret();

    await this.prisma.app.update({
      where: { id: appId },
      data: {
        appKey: newAppKey,
        appSecret: this.cryptoService.hash(newAppSecret),
      },
    });

    return {
      appKey: newAppKey,
      appSecret: newAppSecret,
    };
  }

  /**
   * 生成应用密钥
   */
  private generateAppKey(): string {
    const prefix = 'APP';
    const timestamp = Date.now().toString(36);
    const random = Math.random().toString(36).substr(2, 12);
    return `${prefix}${timestamp}${random}`.toUpperCase();
  }

  /**
   * 生成应用密钥
   */
  private generateAppSecret(): string {
    const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789';
    let secret = '';
    for (let i = 0; i < 32; i++) {
      secret += chars.charAt(Math.floor(Math.random() * chars.length));
    }
    return secret;
  }

  /**
   * 验证应用签名
   */
  async verifySignature(
    appKey: string,
    timestamp: string,
    nonce: string,
    signature: string,
  ): Promise<boolean> {
    const app = await this.prisma.app.findFirst({
      where: { appKey, status: 1, isDeleted: 0 },
    });

    if (!app) {
      return false;
    }

    // 生成签名
    const appSecret = app.appSecret; // 需要解密
    const params = [appSecret, timestamp, nonce].sort().join('');
    const expectedSignature = this.cryptoService.sha256(params);

    return expectedSignature === signature;
  }
}
```

```typescript
          name: true,
          code: true,

          logo: true,
          website: true,

          installCount: true,
          rating: true,
          createdAt: true,
        },
      }),
      this.prisma.app.count({ where }),
    ]);

      total,
      page,
      limit,
      totalPages: Math.ceil(total / limit),
    };
  }

   * 获取应用详情

    }

    // 增加访问量

      data: { viewCount: { increment: 1 } },
    });

    return app;
  }

   * 安装应用
```

```typescript
        _sum: { duration: true },
      }),
      this.prisma.appErrorLog.groupBy({
        by: ['errorType'],
        where: { appId, createdAt: { gte: startDate } },
        _count: true,
      }),
    ]);

    // 总览
    const totalAccess = accessStats.reduce((sum, item) => sum + item._count, 0);
    const totalErrors = errorStats.reduce((sum, item) => sum + item._count, 0);

    return {
      totalAccess,
      totalErrors,
      errorRate: totalAccess > 0 ? (totalErrors / totalAccess * 100).toFixed(2) : '0',
      avgDuration: accessStats.find(s => s.status === 200)?._sum?.duration / 
                   (accessStats.find(s => s.status === 200)?._count || 1) || 0,
      statusBreakdown: accessStats.map(s => ({
        status: s.status,
        count: s._count,
      })),
      errorBreakdown: errorStats.map(s => ({
        type: s.errorType,
        count: s._count,
      })),
    };
  }

  /**
   * 获取开始日期
   */
  private getStartDate(period: 'day' | 'week' | 'month'): Date {
    const now = new Date();
    switch (period) {
      case 'day':
        return new Date(now.getFullYear(), now.getMonth(), now.getDate() - 1);
      case 'week':
        return new Date(now.getTime() - 7 * 24 * 60 * 60 * 1000);
      case 'month':
        return new Date(now.getFullYear(), now.getMonth() - 1, 1);
      default:
        return new Date(now.getTime() - 24 * 60 * 60 * 1000);
    }
```

```typescript
// common/services/encryption.service.ts
import { Injectable } from '@nestjs/common';
import * as crypto from 'crypto';

@Injectable()
export class EncryptionService {
  private readonly ALGORITHM = 'aes-256-gcm';
  private readonly KEY_LENGTH = 32;
  private readonly IV_LENGTH = 12;

  private getEncryptionKey(): Buffer {
    const key = process.env.ENCRYPTION_KEY;
    if (!key) {
      throw new Error('ENCRYPTION_KEY environment variable is required');
    }
    return Buffer.from(key, 'hex');
  }

  /**
   * 加密敏感字段
   */
  encrypt(plaintext: string, context?: string): string {
    const key = this.getEncryptionKey();
    const iv = crypto.randomBytes(this.IV_LENGTH);
    
    const cipher = crypto.createCipheriv(this.ALGORITHM, key, iv);
    
    let encrypted = cipher.update(plaintext, 'utf8', 'hex');
    encrypted += cipher.final('hex');
    
    const authTag = cipher.getAuthTag();
    
    let result = iv.toString('hex') + ':' + encrypted + ':' + authTag.toString('hex');
    
    if (context) {
      result += ':' + Buffer.from(context).toString('hex');
    }
    
    return result;
  }

  /**
   * 解密敏感字段
   */
  decrypt(encryptedData: string, context?: string): string {
    const key = this.getEncryptionKey();
    const parts = encryptedData.split(':');
    
    if (parts.length < 3) {
      throw new Error('Invalid encrypted data format');
    }

    const iv = Buffer.from(parts[0], 'hex');
    const encryptedText = parts[1];
    const authTag = Buffer.from(parts[2], 'hex');
    
    let providedContext = parts[3] ? Buffer.from(parts[3], 'hex').toString() : '';

    if (context && providedContext && context !== providedContext) {
      throw new Error('Context mismatch');
    }

    const decipher = crypto.createDecipheriv(this.ALGORITHM, key, iv);
    decipher.setAuthTag(authTag);

    let decrypted = decipher.update(encryptedText, 'hex', 'utf8');
    decrypted += decipher.final('utf8');

    return decrypted;
  }

  /**
   * 生成哈希（不可逆）
   */
  hash(data: string, salt?: string): string {
    const saltValue = salt || process.env.HASH_SALT || '';
    return crypto.createHmac('sha256', saltValue)
      .update(data)
      .digest('hex');
  }
}
```

```typescript
// common/decorators/sensitive.decorator.ts
import { SetMetadata } from '@nestjs/common';

export const SENSITIVE_KEY = 'sensitive';
export const Sensitive = (strategy: 'phone' | 'idcard' | 'email' | 'bankcard' = 'phone') =>
  SetMetadata(SENSITIVE_KEY, strategy);

// common/interceptors/sensitive.interceptor.ts
import {
  Injectable,
  NestInterceptor,
  ExecutionContext,
  CallHandler,
} from '@nestjs/common';
import { Observable } from 'rxjs';
import { map } from 'rxjs/operators';
import { SENSITIVE_KEY } from '@/common/decorators/sensitive.decorator';

@Injectable()
export class SensitiveInterceptor implements NestInterceptor {
  intercept(context: ExecutionContext, next: CallHandler): Observable<any> {
    return next.handle().pipe(
      map(data => this.sanitize(data)),
    );
  }

  private sanitize(obj: any): any {
    if (!obj || typeof obj !== 'object') {
      return obj;
    }

    if (Array.isArray(obj)) {
      return obj.map(item => this.sanitize(item));
    }

    const result: any = {};
    for (const key in obj) {
      if (obj.hasOwnProperty(key)) {
        const metadata = Reflect.getMetadata(SENSITIVE_KEY, obj, key);
        if (metadata) {
          result[key] = this.maskValue(obj[key], metadata);
        } else {
          result[key] = this.sanitize(obj[key]);
        }
      }
    }
    return result;
  }

  private maskValue(value: string, strategy: string): string {
    if (!value || typeof value !== 'string') {
      return value;
    }

    switch (strategy) {
      case 'phone':
        // 138****0000
        return value.replace(/(\d{3})\d{4}(\d{4})/, '$1****$2');
      case 'idcard':
        // 110***********1234
        return value.replace(/(\d{3})\d*(\d{4})/, '$1*********$2');
      case 'email':
        // abc***@example.com
        const [local, domain] = value.split('@');
        return local.length > 2 
          ? `${local.substring(0, 2)}***@${domain}`
          : value;
      case 'bankcard':
        // 622202*********1234
        return value.replace(/(\d{4})\d*(\d{4})/, '$1*********$2');
      default:
        return value;
    }
  }
}
```

```typescript
      // 执行mysqldump

        '-u', process.env.DATABASE_USER,
        `-p${process.env.DATABASE_PASSWORD}`,
        process.env.DATABASE_NAME,
        '--single-transaction',
        '--quick',
        '--lock-tables=false',
      ]);

      // 压缩
      const gzip = zlib.createGzip();
      const outfile = fs.createWriteStream(backupPath);

      mysqldump.stdout.pipe(gzip).pipe(outfile);

      await new Promise((resolve, reject) => {
        outfile.on('finish', resolve);
        outfile.on('error', reject);
      });

      // 清理过期备份
      await this.cleanupOldBackups();

      // 上传到云存储（如果配置）
      if (this.config.get('BACKUP_REMOTE_URL')) {
        await this.uploadToRemote(backupPath, backupFile);
      }

      return { success: true, file: backupFile };
    } catch (error) {
      throw new BusinessException(5000, `备份失败: ${error.message}`);
    }
  }

  /**
   * 增量备份
   */
  async incrementalBackup() {
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
    const backupFile = `backup-incremental-${timestamp}.sql.gz`;
    const backupPath = path.join(this.BACKUP_DIR, backupFile);

    // 获取binlog位置
    const { stdout } = await this.exec('mysql', [
      '-u', process.env.DATABASE_USER,
      `-p${process.env.DATABASE_PASSWORD}`,
      '-e', 'SHOW MASTER STATUS',

    const execute = childProcess.spawn('mysqlbinlog', [

      `--stop-datetime=${new Date().toISOString()}`,
      binlogInfo.file,
    ]);

    // ... 压缩和保存逻辑
  }

  /**
   * 数据恢复
   */
  async restore(backupFile: string): Promise<void> {
    const backupPath = path.join(this.BACKUP_DIR, backupFile);

    if (!fs.existsSync(backupPath)) {
      throw new BusinessException(404, '备份文件不存在');
    }

    // 解压
    const gunzip = zlib.createGunzip();
    const infile = fs.createReadStream(backupPath);

    const mysql = childProcess.spawn('mysql', [
      '-u', process.env.DATABASE_USER,
      `-p${process.env.DATABASE_PASSWORD}`,
      process.env.DATABASE_NAME,
    ]);

    infile.pipe(gunzip).pipe(mysql.stdin);

    await new Promise((resolve, reject) => {
      mysql.on('close', code => {
        if (code === 0) resolve();
        else reject(new Error('Restore failed'));
      });
    });
  }
```

```typescript
// common/utils/diff.util.ts
export class DiffUtil {
  /**
   * 对比两个对象的差异
   */
  static diff(obj1: any, obj2: any): Record<string, { old: any; new: any }> {
    const changes: Record<string, { old: any; new: any }> = {};

    const keys = new Set([...Object.keys(obj1 || {}), ...Object.keys(obj2 || {})]);
    
    for (const key of keys) {
      const val1 = obj1?.[key];
      const val2 = obj2?.[key];

      if (JSON.stringify(val1) !== JSON.stringify(val2)) {
        changes[key] = { old: val1, new: val2 };
      }
    }

    return changes;
  }

  /**
   * 格式化变更差异
   */
  static format(changes: Record<string, { old: any; new: any }>): string {
    return Object.entries(changes).map(([key, { old, new: newValue }]) => {
      return `${key}: ${JSON.stringify(old)} => ${JSON.stringify(newValue)}`;
    }).join('; ');
  }

  /**
   * 过滤敏感字段
   */
  static filterSensitive(obj: any, sensitiveFields: string[] = []): any {
    if (!obj || typeof obj !== 'object') {
      return obj;
    }

    const result: any = Array.isArray(obj) ? [] : {};
    
    for (const key in obj) {
      if (obj.hasOwnProperty(key)) {
        if (sensitiveFields.includes(key)) {
          result[key] = '***'; // 脱敏
        } else {
          result[key] = this.filterSensitive(obj[key], sensitiveFields);
        }
      }
    }

    return result;
  }
}
```

```typescript
// common/guards/api-version.guard.ts
import { Injectable, CanActivate, ExecutionContext } from '@nestjs/common';

@Injectable()
export class ApiVersionGuard implements CanActivate {
  private readonly supportedVersions = ['v1', 'v2'];
  private readonly defaultVersion = 'v1';

  canActivate(context: ExecutionContext): boolean {
    const request = context.switchToHttp().getRequest();
    const version = this.extractVersion(request);

    if (!version || !this.supportedVersions.includes(version)) {
      request.params.version = this.defaultVersion;
      return true;
    }

    request.params.version = version;
    return true;
  }

  private extractVersion(request: any): string | null {
    // 从URL路径提取版本: /api/v1/users
    if (request.path.startsWith('/api/v1/')) return 'v1';
    if (request.path.startsWith('/api/v2/')) return 'v2';

    // 从Header提取版本: X-API-Version: v2
    const headerVersion = request.headers['x-api-version'];
    if (headerVersion && this.supportedVersions.includes(headerVersion)) {
      return headerVersion;
    }

    // 从Query参数提取版本: /api/users?version=v2
    const queryVersion = request.query.version;
    if (queryVersion && this.supportedVersions.includes(queryVersion)) {
      return queryVersion;
    }

    return null;
  }
}
```

```typescript
    const max = 100;

    const current = await this.redis.incr(key);
    if (current === 1) {
      await this.redis.expire(key, windowMs / 1000);
    }

    // 设置响应头
    res.setHeader('X-RateLimit-Limit', max);
    res.setHeader('X-RateLimit-Remaining', Math.max(0, max - current));
    res.setHeader('X-RateLimit-Reset', new Date(Date.now() + windowMs).toISOString());

    if (current > max) {
      return res.status(429).json({
        code: 42900,
        message: '请求过于频繁，请稍后重试',
        retryAfter: windowMs,
      });
    }

    next();
  }
}
```

```typescript
// modules/webhook/services/webhook.service.ts
import { Injectable, BusinessException } from '@/common';
import { Cron, CronExpression } from '@nestjs/schedule';
import * as crypto from 'crypto';

@Injectable()
export class WebhookService {
  private readonly SIGNATURE_HEADER = 'X-Signature';
  private readonly TIMESTAMP_HEADER = 'X-Timestamp';
  private readonly TOLERANCE = 300000; // 5分钟

  /**
   * 验证Webhook签名
   */
  verifySignature(
    payload: string,
    signature: string,
    timestamp: string,
    secret: string,
  ): boolean {
    // 检查时间戳
    const now = Date.now();
    const receivedTime = parseInt(timestamp, 10);
    if (Math.abs(now - receivedTime) > this.TOLERANCE) {
      throw new BusinessException(1000, 'Webhook已过期');
    }

    // 验证签名
    const expectedSignature = crypto
      .createHmac('sha256', secret)
      .update(`${timestamp}${payload}`)
      .digest('hex');

    return crypto.timingSafeEqual(
      Buffer.from(signature),
      Buffer.from(expectedSignature),
    );
  }

  /**
   * 发送Webhook
   */
  async sendWebhook(
    url: string,
    payload: any,
    secret: string,
  ): Promise<{ success: boolean; statusCode?: number; error?: string }> {
    const timestamp = Date.now().toString();
    const body = JSON.stringify(payload);
    const signature = crypto
      .createHmac('sha256', secret)
      .update(`${timestamp}${body}`)
      .digest('hex');

    for (let attempt = 0; attempt < 3; attempt++) {
      try {
        const response = await fetch(url, {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
            [this.SIGNATURE_HEADER]: signature,
            [this.TIMESTAMP_HEADER]: timestamp,
          },
          body: body,
          timeout: 5000,
        });

        if (response.ok) {
          return { success: true, statusCode: response.status };
        }
      } catch (error) {
        if (attempt === 2) {
          return { success: false, error: error.message };
        }
        await this.delay(Math.pow(2, attempt) * 1000);
      }
    }

    return { success: false };
  }

  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }

  /**
   * 创建Webhook订阅
   */
  async createSubscription(
    userId: number,
    event: string,
    callbackUrl: string,
    secret: string,
  ) {
    // 创建订阅记录
  }

  /**
   * 发送事件到所有订阅者
   */
  async dispatchEvent(event: string, payload: any) {
    const subscriptions = await this.prisma.webhookSubscription.findMany({
      where: { event, status: 1, isDeleted: 0 },
    });

    for (const sub of subscriptions) {
      this.sendWebhook(sub.callbackUrl, payload, sub.secret).catch(err => {
        this.logger.error(`Webhook发送失败: ${sub.callbackUrl}`, err);
      });
    }
  }
}
```

```typescript
// common/interceptors/audit.interceptor.ts
import { Injectable, NestInterceptor, ExecutionContext, CallHandler } from '@nestjs/common';
import { Observable } from 'rxjs';
import { map } from 'rxjs/operators';

@Injectable()
export class AuditInterceptor implements NestInterceptor {
  intercept(context: ExecutionContext, next: CallHandler): Observable<any> {
    const request = context.switchToHttp().getRequest();
    const user = request.user;

    // 记录请求开始时间
    const startTime = Date.now();

    return next.handle().pipe(
      map(data => {
        const duration = Date.now() - startTime;
        const status = request.res?.statusCode || 200;

        // 仅记录写操作
        if (['POST', 'PUT', 'PATCH', 'DELETE'].includes(request.method)) {
          this.auditLogService.create({
            userId: user?.id,
            username: user?.username,
            module: this.getModule(request),
            entity: this.getEntity(request),
            operation: this.getOperation(request.method),
            targetId: data?.id || request.params?.id,
            oldValues: this.getOldValues(request),
            newValues: this.getNewValues(request, data),
            ip: request.ip,
            userAgent: request.headers['user-agent'],
            browser: this.getBrowser(request),
            os: this.getOS(request),
            status: status >= 400 ? 2 : 1,
            errorMsg: status >= 400 ? '操作失败' : null,
            requestId: request.headers['x-request-id'],
          });
        }

        return data;
      }),
    );
  }

  private getModule(request: any): string {
    return request.route?.config?.module || 'unknown';
  }

  private getEntity(request: any): string {
    return request.route?.config?.entity || 'unknown';
  }

  private getOperation(method: string): string {
    const map: Record<string, string> = {
      POST: 'create',
      PUT: 'update',
      PATCH: 'partial_update',
      DELETE: 'delete',
    };
    return map[method] || 'other';
  }

  private getOldValues(request: any): string {
    if (request.method !== 'PATCH') return null;
    // 获取数据库中的原始值
    return JSON.stringify(request.originalData || {});
  }

  private getNewValues(request: any, result: any): string {
    const body = { ...request.body };
    if (result) {
      Object.assign(body, result);
    }
    return JSON.stringify(body);
  }

  private getBrowser(request: any): string {
    const ua = request.headers['user-agent'] || '';
    if (ua.includes('Chrome')) return 'Chrome';
    if (ua.includes('Firefox')) return 'Firefox';
    if (ua.includes('Safari')) return 'Safari';
    if (ua.includes('Edge')) return 'Edge';
    return 'Unknown';
  }

  private getOS(request: any): string {
    const ua = request.headers['user-agent'] || '';
    if (ua.includes('Windows')) return 'Windows';
    if (ua.includes('Mac')) return 'Mac';
    if (ua.includes('Linux')) return 'Linux';
    if (ua.includes('iOS')) return 'iOS';
    if (ua.includes('Android')) return 'Android';
    return 'Unknown';
  }
}
```

```typescript
// common/circuit-breaker/circuit-breaker.service.ts
import { Injectable } from '@nestjs/common';
import { CircuitBreaker } from 'opossum';

@Injectable()
export class CircuitBreakerService {
  private breakers: Map<string, CircuitBreaker> = new Map();

  getBreaker(name: string, options: any = {}): CircuitBreaker {
    if (!this.breakers.has(name)) {
      const breaker = new CircuitBreaker(async (payload: any) => {
        return options.executor(payload);
      }, {
        timeout: options.timeout || 3000,
        errorThreshold: options.errorThreshold || 5,
        resetTimeout: options.resetTimeout || 30000,
        halfOpenRequests: options.halfOpenRequests || 1,
        name,
      });

      breaker.on('halfOpen', () => {
        console.log(`Circuit breaker ${name} is half-open`);
      });

      breaker.on('close', () => {
        console.log(`Circuit breaker ${name} is closed`);
      });

      breaker.on('open', () => {
        console.log(`Circuit breaker ${name} is open`);
      });

      this.breakers.set(name, breaker);
    }

    return this.breakers.get(name);
  }

  async execute<T>(
    name: string,
    executor: () => Promise<T>,
    fallback?: () => Promise<T>,
  ): Promise<T> {
    const breaker = this.getBreaker(name, {
      executor: () => executor(),
      timeout: 5000,
      errorThreshold: 5,
    });

    if (fallback) {
      breaker.fallback(fallback);
    }

    return breaker.fire();
  }
}
```

#### 限流桶算法
```typescript
// common/rate-limit/token-bucket.service.ts
import { Injectable } from '@nestjs/common';

@Injectable()
export class TokenBucketService {
  private buckets: Map<string, { tokens: number; lastRefill: number }> =
    new Map();

  constructor(private readonly redis: RedisService) {}

  async consume(key: string, tokens: number, maxTokens: number, refillRate: number): Promise<boolean> {
    const bucketKey = `tokenbucket:${key}`;
    const data = await this.redis.get(bucketKey);

    let bucket: { tokens: number; lastRefill: number };
    if (data) {
      bucket = JSON.parse(data);
    } else {
      bucket = { tokens: maxTokens, lastRefill: Date.now() };
    }

    // 补充令牌
    const now = Date.now();
    const tokensToFill = (now - bucket.lastRefill) / 1000 * refillRate;
    bucket.tokens = Math.min(maxTokens, bucket.tokens + tokensToFill);
    bucket.lastRefill = now;

    if (bucket.tokens >= tokens) {
      bucket.tokens -= tokens;
      await this.redis.set(bucketKey, JSON.stringify(bucket), 'EX', 3600);
      return true;
    }

    await this.redis.set(bucketKey, JSON.stringify(bucket), 'EX', 3600);
```

### 24.4 数据备份与恢复

#### 备份策略
```typescript
// modules/backup/services/backup-strategy.service.ts
import { Injectable } from '@nestjs/common';

@Injectable()
export class BackupStrategyService {
  /**
   * 全量备份策略
   */
  async fullBackup() {
    const timestamp = new Date().toISOString().split('T')[0];
    const backupName = `full-backup-${timestamp}`;

    // 锁定数据库
    await this.prisma.$executeRaw`FLUSH TABLES WITH READ LOCK`;

    try {
      // 备份数据
      const backup = await this.databaseBackupService.backup();

      // 上传到对象存储
      await this.storageService.upload(backupName, backup);

      // 记录备份信息
      await this.prisma.backupRecord.create({
        data: {
          type: 'full',
          fileName: backupName,
          status: 1,
          fileSize: backup.size,
        },
      });
    } finally {
      // 解锁数据库
      await this.prisma.$executeRaw`UNLOCK TABLES`;
    }
  }

  /**
   * 增量备份策略
   */
  async incrementalBackup() {
    const lastBackup = await this.prisma.backupRecord.findFirst({
      where: { type: 'full' },
      orderBy: { createdAt: 'desc' },
    });

    const binlogPosition = lastBackup?.binlogPosition || '0';

    const增量备份 = await this.databaseBackupService.backupBinlog(binlogPosition);

    await this.storageService.upload(`incremental-${Date.now()}`, 增量备份);
  }

  /**
   * 数据恢复
   */
  async restore(backupName: string): Promise<void> {
    const backup = await this.storageService.download(backupName);

    // 停止应用服务
    await this.appService.stop();

    try {
      // 恢复数据
      await this.databaseBackupService.restore(backup);

      // 重启应用服务
      await this.appService.start();
    } catch (error) {
      // 恢复失败，回滚
      await this.databaseBackupService.restore(this.lastBackup);
      throw error;
    }
  }
}
```

```typescript
@Injectable()
export class AuthService {
  async login(dto: LoginDto): Promise<LoginResult> {
    // 1. 验证用户名密码
    const user = await this.prisma.baseUser.findUnique({
      where: { username: dto.username },
      include: { userTenants: { include: { tenant: true } } },

      username: user.username,

      tenantId: ut.tenant.id,
      tenantName: ut.tenant.name,
      tenantCode: ut.tenant.code,
      role: ut.role,
      isDefault: ut.isDefault,
    }));

    return { token, teams, user: this.transformUser(user) };
  }
}
```

```typescript
@Injectable()
export class TenantInterceptor implements NestInterceptor {
  intercept(context: ExecutionContext, next: CallHandler): Observable<any> {
    const req = context.switchToHttp().getRequest();
    const user = req.user;

    // 从 JWT 获取用户ID

    req.context = {
      tenantId,
      businessCode,
      appCode,
      userId,
```

### 27.6 权限检查流程

#### 流程说明
```
1. 用户访问菜单/接口：/api/v1/users
2. 验证租户上下文
3. 获取用户角色（通过 MapUserRole）
4. 获取角色的权限组（通过 BasePermissionGroup）
5. 获取权限组的权限（通过 MapPermissionGroupPermission）
6. 检查权限码是否匹配：'user:list'
7. 放行或返回 403
```

#### 代码示例
```typescript
// common/guards/permission.guard.ts
@Injectable()
export class PermissionGuard implements CanActivate {
  async canActivate(context: ExecutionContext): Promise<boolean> {
    const req = context.switchToHttp().getRequest();
    const user = req.user;
    const targetPermission = req.route?.config?.permission;

    if (!targetPermission) {
      return true; // 无权限要求
    }

    // 获取用户权限列表
    const permissions = await this.getUserPermissions(user.id, req.context.tenantId);

    // 检查是否拥有目标权限
    const hasPermission = permissions.some(
      p => p.code === targetPermission || p.code.startsWith(targetPermission + ':')
    );

    if (!hasPermission) {
      throw new BusinessException(1003, '无权访问');
    }

    return true;
  }

  private async getUserPermissions(userId: number, tenantId: number): Promise<any[]> {
    return this.prisma.$queryRaw`
      SELECT DISTINCT p.code
      FROM base_users u
      JOIN user_tenants ut ON u.id = ut.user_id
      JOIN map_user_role mur ON u.id = mur.user_id
      JOIN base_role r ON mur.role_id = r.id
      JOIN base_permission_groups rpg ON r.id = rpg.role_id
      JOIN map_permission_group_permission mpgp ON rpg.id = mpgp.permission_group_id
      JOIN base_permissions p ON mpgp.permission_id = p.id
      WHERE u.id = ${userId}
        AND ut.tenant_id = ${tenantId}
        AND ut.is_deleted = 0
        AND r.status = 1
        AND p.status = 1;
    `;
  }
}
```

```typescript
// modules/app/services/app.service.ts
@Injectable()
export class AppService {
  async create(tenantId: number, dto: AppCreateDto): Promise<App> {
    // 1. 验证业务存在
    const business = await this.prisma.business.findUnique({
      where: { id: dto.businessId },
    });
    if (!business) {
      throw new BusinessException(2002, '业务不存在');
    }

    // 2. 验证应用代码唯一性（在业务范围内）
    const exists = await this.prisma.app.findFirst({
      where: {
        businessId: dto.businessId,
        code: dto.code,
        isDeleted: 0,
      },
    });
    if (exists) {
      throw new BusinessException(2001, '应用代码已存在');
    }

    // 3. 创建应用
    const app = await this.prisma.app.create({
      data: {
        name: dto.name,
        code: dto.code,
        businessId: dto.businessId,
        route: dto.route,
        icon: dto.icon,
        status: 1,
      },
    });

    // 4. 生成应用密钥
    const appKey = `APP_${business.code}_${dto.code}_${Date.now().toString(36).toUpperCase()}`;
    const appSecret = crypto.randomBytes(32).toString('hex');

    // 5. 创建应用密钥记录
    await this.prisma.appSecret.create({
      data: {
        appId: app.id,
        appKey,
        appSecret: this.crypto.hash(appSecret),
      },
    });

    // 6. 创建默认权限组
    await this.createDefaultPermissions(app.id, dto.businessId);

    return { ...app, appSecret }; // 只返回一次密钥
  }
}
```

### 27.8 数据导出流程（带三元组）

#### 流程说明
```
1. 用户选择数据范围（租户+业务+应用）
2. 点击"导出"按钮
3. POST /api/v1/export/create
4. 创建 ExportJob 记录（状态：pending）
5. 异步任务处理导出
6. 查询数据时带上三元组条件
7. 生成 Excel 文件
8. 上传到对象存储
9. 更新 ExportJob 状态为 completed
10. 发送通知给用户
```

#### 代码示例
```typescript
// modules/export/services/export.service.ts
@Injectable()
export class ExportService {
  async createExport(
    userId: number,
    tenantId: number,
    businessCode: string,
    appCode: string,
    type: string,
    filters: any,
  ): Promise<ExportJob> {
    // 1. 验证租户权限
    await this.verifyTenantPermission(userId, tenantId);

    // 2. 创建导出任务
    const job = await this.prisma.exportJob.create({
      data: {
        userId,
        tenantId,
        businessCode,
        appCode,
        type,
        filters: JSON.stringify(filters),
        status: 'pending',
      },
    });

    // 3. 异步处理
    this.queue.add('export-process', { jobId: job.id });

    return job;
  }

  async processExport(jobId: number): Promise<void> {
    const job = await this.prisma.exportJob.findUnique({
      where: { id: jobId },
    });

    // 4. 查询数据（带三元组隔离）
    const data = await this.prisma.product.findMany({
      where: {
        tenantId: job.tenantId,
        businessCode: job.businessCode,
        appCode: job.appCode,
        ...this.parseFilters(job.filters),
      },
    });

    // 5. 生成 Excel
    const filePath = await this.generateExcel(data);

    // 6. 上传存储
    const fileUrl = await this.storage.upload(filePath);

    // 7. 更新任务状态
    await this.prisma.exportJob.update({
      where: { id: jobId },
      data: {
        status: 'completed',
        filePath: fileUrl,
        completedAt: new Date(),
      },
    });
  }
}
```

## 附录：多租户多业务架构设计

### 架构概述

本系统采用**三层架构设计**：

```
用户(BaseUser) → 租户(Tenant) → 业务(Business) → 应用(App)
```

### 数据模型关系

#### 1. UserTenant 表（用户-租户关联）

```prisma
model UserTenant {
  id        BigInt   @id @default(autoincrement()) @map("id")
  userId    BigInt   @map("user_id")
  tenantId  BigInt   @map("tenant_id")
  role      String   @default("member") @map("role") @db.VarChar(20)
  isDefault Int      @default(0) @map("is_default")
  createdAt DateTime @default(now()) @map("created_at")

  user   BaseUser @relation(fields: [userId], references: [id])
  tenant Tenant   @relation(fields: [tenantId], references: [id])

  @@unique([userId, tenantId])
  @@index([tenantId])
  @@map("user_tenants")
}
```

**功能说明**：
- 实现用户对多个租户的多对多访问
- `isDefault` 字段标记用户的默认租户
- `role` 字段定义用户在租户中的角色（member/admin）

#### 2. TenantBusiness 表（租户-业务关联）

```prisma
model TenantBusiness {
  id         BigInt   @id @default(autoincrement()) @map("id")
  tenantId   BigInt   @map("tenant_id")
  businessId BigInt   @map("business_id")
  status     Int      @default(1) @map("status")
  expiresAt  DateTime? @map("expires_at")
  createdAt  DateTime @default(now()) @map("created_at")

  tenant   Tenant   @relation(fields: [tenantId], references: [id])
  business Business @relation(fields: [businessId], references: [id])

  @@unique([tenantId, businessId])
  @@index([tenantId])
  @@index([businessId])
  @@map("tenant_businesses")
}
```

**功能说明**：
- 实现租户对多个业务的多对多订阅
- `status` 字段标记订阅状态（1=启用，0=禁用）
- `expiresAt` 字段支持订阅过期时间

### 租户切换流程

```
1. 用户登录 → 获取所有可访问租户（通过 UserTenant）
2. 用户选择租户 → 将 tenantId 存入 Token/Session
3. 请求携带租户ID → 通过 Guard 验证访问权限
4. 数据查询 → 自动添加 tenantId 过滤条件
```

### 业务切换流程

```
1. 租户管理员订阅业务 → 创建 TenantBusiness 记录
2. 用户切换业务 → 修改当前租户的 businessCode
3. 请求携带业务Code → 通过 Guard 验证订阅状态
4. 数据查询 → 自动添加 businessCode 过滤条件
```

## 目录说明

### /docs - 文档目录
所有项目文档集中存放，包括框架文档、认证文档、更新日志等。

### /src - 源代码目录
- **common/** 公共组件，被所有模块依赖
- **infrastructure/** 基础设施服务（数据库、缓存等）
- **modules/** 业务模块目录
- **types/** 全局类型定义
- **config/** 配置文件

### /prisma - 数据库相关
- **schema.prisma** Prisma模型定义（核心）
- **migrations/** 数据库迁移文件

### /src/modules/system - 系统模块
已实现的系统功能模块：
- **auth** - 认证（登录、Token、RSA加密）
- **tenant** - 租户管理
- **business** - 业务线管理
- **app** - 应用模块管理
- **user** - 用户管理
- **role** - 角色管理
- **permission** - 权限管理
- **org** - 组织架构
- **config** - 系统配置
- **dict** - 字典管理
- **captcha** - 验证码

## 分层架构

```
┌─────────────────────────────────────────────────────────────┐
│                    表现层 (Presentation)                      │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  Controller (控制器)                                    │  │
│  │  - 接收请求                                              │  │
│  │  - 参数校验                                              │  │
│  │  - 调用 Service                                          │  │
│  └────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                    应用层 (Application)                       │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  Service (服务)                                         │  │
│  │  - 业务逻辑编排                                          │  │
│  │  - 事务管理                                              │  │
│  │  - 调用 Repository                                       │  │
│  └────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                    领域层 (Domain)                            │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  DTO (数据传输对象)                                     │  │
│  │  - 创建数据验证                                          │  │
│  │  - 更新数据验证                                          │  │
│  └────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                   基础设施层 (Infrastructure)                  │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  Repository (数据访问)                                  │  │
│  │  - Prisma 操作                                           │  │
│  │  - 缓存操作                                              │  │
│  └────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

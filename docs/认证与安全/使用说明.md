# 滑块验证码实现说明

## 核心改进

### 1. 响应格式简化

**之前的设计**：前端需要处理复杂的图片URL，二次请求图片资源

**现在的设计**：直接返回 Base64 图片数据，前端可以直接使用

```typescript
{
  captchaId: "uuid",
  y: 45,              // ← 只需返回Y轴位置
  correctX: 0,        // ← X轴从0开始，固定返回0
  shape: "circle",    // ← 形状类型
  backgroundImage: "data:image/png;base64,iVBORw0KGgoAAAANS...", // 背景图
  sliderImage: "data:image/png;base64,iVBORw0KGgoAAAANS...",     // 滑块图
  width: 310,
  height: 155,
  blockSize: 40
}
```

### 2. Y轴位置原理

```
┌─────────────────────────────────────────────────────────┐
│  背景图 (310×155)                                        │
│  ┌───────────────────────────────────────────────────┐  │
│  │  纵向随机位置 Y=45px                              │  │
│  │                                                   │  │
│  │  ┌─────────────────┐                              │  │
│  │  │  滑块需移动到   │                              │  │
│  │  │  Y=45px 位置   │                              │  │
│  │  └─────────────────┘                              │  │
│  │                                                   │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘

前端只需要：
1. 在 Y=45px 位置显示滑块
2. 水平向右拖动滑块
3. 拖动距离 X 即为验证结果
```

### 4. 前端使用示例

```vue
<template>
  <div class="captcha-wrapper">
    <!-- 背景图 -->
    <div class="captcha-bg" :style="bgStyle">
      <!-- 滑块 -->
      <div
        class="slider"
        :style="sliderStyle"
        @mousedown="handleMouseDown"
      >
        <span>拖动</span>
      </div>
    </div>
    
    <button @click="refreshCaptcha">刷新验证码</button>
    <button @click="submitLogin" :disabled="!captchaVerified">登录</button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

// 验证码数据
const captchaData = ref(null)
const sliderLeft = ref(0)
const isDragging = ref(false)
const captchaVerified = ref(false)

// 背景图样式 - 使用Base64图片
const bgStyle = computed(() => ({
  width: '310px',
  height: '155px',
  backgroundImage: captchaData.value 
    ? `url(${captchaData.value.backgroundImage})` 
    : 'none',
  backgroundPosition: `0 -${captchaData.value?.y}px`,
  backgroundSize: '310px 155px',
  position: 'relative',
  overflow: 'hidden',
  marginBottom: '10px'
}))

// 滑块样式 - 只在Y轴位置显示
const sliderStyle = computed(() => ({
  width: '44px',
  height: '44px',
  backgroundImage: captchaData.value 
    ? `url(${captchaData.value.sliderImage})` 
    : 'none',
  position: 'absolute',
  left: `${sliderLeft.value}px`,
  top: `${captchaData.value?.y}px`, // ← 在Y轴位置显示
  cursor: 'grab',
  backgroundRepeat: 'no-repeat'
}))

// 获取验证码
const getCaptcha = async () => {
  const res = await fetch('/api/v1/auth/captcha/slider')
  captchaData.value = await res.json()
  sliderLeft.value = 0
  captchaVerified.value = false
}

// 鼠标按下
const handleMouseDown = (e) => {
  isDragging.value = true
  startPos.value = e.clientX
}

// 鼠标移动
const handleMouseMove = (e) => {
  if (!isDragging.value) return
  const deltaX = e.clientX - startPos.value
  sliderLeft.value = Math.max(0, Math.min(deltaX, 270)) // 310 - 44 = 266
}

// 鼠标松开
const handleMouseUp = async () => {
  if (!isDragging.value) return
  isDragging.value = false
  
  const currentX = sliderLeft.value
  const t = Date.now() - startTime.value
  
  // 验证
  const res = await fetch('/api/v1/auth/captcha/slider/verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      captchaId: captchaData.value.captchaId,
      captchaResult: JSON.stringify({
        y: captchaData.value.y,     // ← 返回Y轴位置
        t: t,
        track: [] // 可选：记录轨迹
      })
    })
  })
  
  const result = await res.json()
  if (result.valid) {
    captchaVerified.value = true
    alert('验证通过')
  } else {
    alert('验证失败')
    getCaptcha() // 刷新验证码
  }
}

// 刷新验证码
const refreshCaptcha = () => {
  getCaptcha()
}

// 初始化
getCaptcha()
</script>
```

### 5. 验证逻辑

后端只需要验证 **Y轴位置**：

```typescript
async verifySliderCaptcha(captchaId: string, captchaResult: string) {
  const stored = await this.cache.get<SliderCaptcha>(captchaKey);
  
  // 验证Y轴位置 (允许 5px 误差)
  const { y, t, track } = JSON.parse(captchaResult);
  const diff = Math.abs(y - stored.correctY);
  
  if (diff > 5) {
    return false; // Y轴位置错误
  }
  
  // 验证时间
  if (t < 300 || t > 60000) {
    return false;
  }
  
  // 验证轨迹（Y轴变化应该很小）
  if (track && track.length > 0) {
    const yChanges = Math.max(...track) - Math.min(...track);
    if (yChanges > 10) {
      return false; // Y轴移动过大
    }
  }
  
  return true;
}
```

### 6. 优势总结

| 对比项 | 之前的设计 | 现在的设计 |
|--------|-----------|-----------|
| 图片加载 | 二次请求 | 一次返回 |
| 响应大小 | URL字符串 | Base64数据 |
| 前端处理 | 请求图片 | 直接使用 |
| 网络请求数 | 3次 (验证码+背景+滑块) | 1次 |
| 实现复杂度 | 需要图片服务 | 内嵌图片 |

### 7. 验证码参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `y` | number | Y轴位置 (关键参数) | 45 |
| `correctX` | number | X轴拖动距离 (后端计算) | 0 |
| `shape` | string | 形状类型 | 'circle' |
| `backgroundImage` | string | 背景图Base64 | data:image/png;base64,... |
| `sliderImage` | string | 滑块图Base64 | data:image/png;base64,... |
| `width` | number | 验证码宽度 | 310 |
| `height` | number | 验证码高度 | 155 |
| `blockSize` | number | 滑块大小 | 40 |
